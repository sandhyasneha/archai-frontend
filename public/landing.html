How it works

4 AI agents. 1 production-ready blueprint.

Every blueprint goes through 4 agents in sequence — no output reaches you until the Auditor signs off.

🛡
Agent 01
Gatekeeper
Validates your request is a legitimate cloud infrastructure requirement. Blocks off-topic and harmful prompts before any cost is incurred.
→
🏗
Agent 02
Architect
Designs the resource dependency graph — VPCs, compute, databases, load balancers, storage — and outputs a structured JSON plan.
→
⚙
Agent 03
Engineer
Converts the plan into complete Terraform HCL using official provider modules. References your Knowledge Base to enforce company standards.
→
✅
Agent 04
Auditor
Validates syntax, checks brace matching, and runs security checks. Rejects and retries automatically (max 3 attempts) until the code is clean.

FAQ

Frequently asked questions

What is ArchAI and who is it for? +
ArchAI is an AI-powered cloud architect platform built for SMBs — startup CTOs, IT managers, DevOps leads, and founders who need production-grade cloud infrastructure without hiring expensive consultants or requiring deep Terraform expertise. You describe your system in plain English; ArchAI generates the complete, compliant Terraform code.
How does the 4-agent pipeline work? +
Four AI agents run in sequence for every blueprint: (1) Gatekeeper validates your prompt is a legitimate cloud request, (2) Architect designs the resource plan as structured JSON, (3) Engineer converts the plan into complete Terraform HCL using official provider modules, and (4) Auditor validates syntax and security — retrying up to 3 times if issues are found. You only see output the Auditor has approved.
How is ArchAI different from ChatGPT or GitHub Copilot? +
Generic AI tools generate generic code. ArchAI enforces your specific company standards through the Knowledge Base — your naming conventions, approved instance types, compliance requirements, and budget caps are injected into every blueprint automatically. The output follows your standards exactly, not a generic template. Plus, the Auditor agent validates the code before you see it — ChatGPT doesn't.
What cloud providers are supported? +
ArchAI supports AWS, Azure, and GCP. The free Scout plan supports AWS only. Pro, Team, and Enterprise plans include all three clouds with provider-specific Terraform using official modules (terraform-aws-modules, azurerm, google provider).
Is the generated Terraform actually production-ready? +
ArchAI generates production-grade Terraform that follows official module patterns, enforces security best practices (private subnets for databases, encrypted storage, least-privilege IAM), and applies your compliance requirements. It's a strong starting point — we recommend engineering review before applying to production, as with any infrastructure code.
How does ArchAI enforce compliance? +
Two ways: (1) Built-in guardrails — the Engineer and Auditor agents automatically enforce SOC 2, GDPR, CIS Benchmark L2, and HIPAA best practices in every blueprint. (2) Knowledge Base — you upload your own compliance documents and the agents reference them when generating code, ensuring your specific policies are followed.
